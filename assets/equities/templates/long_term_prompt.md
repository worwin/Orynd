# Long-Term Analysis Prompt

Use this prompt when building `analysis/long_term_analysis.md` and the related snapshot files.

## Goal
Analyze a U.S. stock as a potential long-term investment using current market data, recent financial news, and a Buffett-style focus on strong fundamentals, intrinsic value, and business quality.

## Required sections
- Company overview
- Fundamental analysis
- Technical overview
- Sentiment analysis
- Valuation and intrinsic value estimate
- Scoring system
- Recommendation

## Rules
- Use direct links and reputable sources.
- Keep the tone neutral and analytical.
- Fundamentals lead; short-term factors matter only if they affect the long-term thesis.
