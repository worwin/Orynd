# Source Hierarchy

## Priority order
1. SEC filings
2. Company annual report, shareholder letter, and proxy
3. Earnings release and transcript
4. Investor presentation
5. Reputable financial data and news sources
6. Analyst summaries and sentiment sources

## Rules
- If data conflicts, prefer SEC filings.
- Use market-data vendors for quotes and multiples, not to override filing numbers.
- Sentiment and social chatter may color the narrative, but may not override fundamentals.
