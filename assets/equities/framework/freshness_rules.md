# Equities Freshness Rules

## Core principle
- Filings drive confidence more than same-day price.
- Stale filings or unclear accounting reduce confidence more than stale technicals.

## Maximum age

### 10-K or 20-F
- Valid as the annual base until superseded.
- Confidence must be reduced if the latest annual filing has not been reviewed.

### 10-Q
- Tactical equity decisions should use the latest quarterly filing.
- If the latest quarter is missing, confidence cannot exceed medium.

### 8-K
- Review immediately if material.
- Missing a material 8-K caps confidence at low.

### Proxy / DEF 14A
- Refresh annually for incentives, governance, and insider context.
- Missing proxy review reduces confidence on management quality.

### Price and technicals
- Useful context, but not primary drivers.
- Stale price data lowers timing precision, not core business conviction.

### Sentiment and news
- Use for context only unless the event changes the long-term thesis.

## Confidence caps
- Missing latest annual filing review: confidence cannot exceed medium.
- Missing latest quarterly filing review: confidence cannot exceed medium.
- Missing material filing or unclear accounting treatment: confidence cannot exceed low.
- If valuation depends on assumptions not grounded in filings, confidence must be reduced.
