# Buffett Statement Rules Index

This folder contains one `README.md` per chapter from *How to Read Financial Statements*.
Each chapter file turns the chapter into a machine-usable rule set with buy signals, no-buy signals, and AI translation notes.

## Working Rule
- No single chapter is enough on its own.
- Use the collection as a layered decision framework.
- Prefer long multi-year history over one-year snapshots.
