# C34 - Total Assets and the Return on Total Assets

## Purpose
Strong businesses can produce attractive earnings from a relatively modest asset base, so return on assets helps Buffett test efficiency.

## Source
- PDF: `C34 - Total Assets and the Return on Total Assets.pdf`
- OCR text: `C34 - Total Assets and the Return on Total Assets.txt`

## Buy Signals
- Prefer businesses with good and consistent returns on total assets
- Treat efficient asset use as supporting evidence of a moat
- Use ROA alongside debt and reinvestment needs

## No-Buy Signals
- Avoid asset-heavy businesses that earn little on those assets
- Do not use asset size alone as a sign of strength
- Avoid companies whose asset growth outruns earnings growth

## AI Rules
- IF returns on assets are consistently healthy THEN add positive efficiency evidence
- IF assets expand without matching earnings power THEN downgrade quality
- Use ROA as support rather than the sole trigger
