# C47 - Return on Shareholders' Equity Part One

## Purpose
ROE matters because it measures how effectively management compounds shareholder capital, but the equity base must be understood first.

## Buy Signals
- Use ROE to test how efficiently management uses owners' money
- Prefer businesses that can convert retained earnings into high future earnings
- Pair ROE with equity composition so the result is interpreted correctly

## No-Buy Signals
- Do not use ROE mechanically without understanding treasury stock and leverage
- Avoid businesses with weak returns on shareholder capital
- Avoid assuming high ROE always means a moat

## AI Rules
- Treat ROE as a major Buffett metric, but only after understanding the equity base
- IF management earns weak returns on shareholder capital THEN lower the quality score
- Carry ROE forward into leverage and buyback checks before final judgment
