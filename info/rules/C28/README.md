# C28 - Total Current Assets and the Current Ratio

## Purpose
The current ratio is not a decisive Buffett moat test because many wonderful businesses run below one thanks to strong earning power.

## Source
- PDF: `C28 - Total Current Assets and the Current Ratio.pdf`
- OCR text: `C28 - Total Current Assets and the Current Ratio.txt`

## Buy Signals
- Allow current ratios below one when earning power is strong
- Use current assets together with debt, cash, and profitability
- Recognize that strong businesses can operate with less liquidity cushion

## No-Buy Signals
- Do not reject a great business just because current ratio is below one
- Do not treat a high current ratio as proof of quality
- Avoid ignoring earnings power when interpreting liquidity ratios

## AI Rules
- Treat current_ratio as contextual rather than decisive
- IF current_ratio < 1 but earnings, cash flow, and debt profile are strong THEN do not auto-fail
- Use the ratio more carefully with marginal businesses than with obvious cash generators
