# C17 - Income Before Tax The Number That Warren Uses

## Purpose
Pretax income is central to Buffett's valuation logic because it represents the earning power later capitalized like an equity bond.

## Buy Signals
- Prefer stable and rising pretax income
- Use pretax income to compare economics across companies
- Treat strong pretax income as the base for valuation work

## No-Buy Signals
- Avoid companies with thin or erratic pretax income
- Avoid using distorted after-tax results without understanding pretax earnings
- Do not value a weak business as an equity bond

## AI Rules
- Normalize around pretax income before valuation
- IF pretax income is durable and rising THEN allow valuation analysis to proceed
- IF pretax income is weak or volatile THEN do not model the company as an equity bond
