# C43 - Shareholders' EquityBook Value

## Purpose
Book value matters mainly because it helps Buffett measure how effectively management compounds shareholder capital.

## Source
- PDF: `C43 - Shareholders' EquityBook Value.pdf`
- OCR text: `C43 - Shareholders' EquityBook Value.txt`

## Buy Signals
- Use book value as the base for return-on-equity analysis
- Prefer businesses that grow equity value through retained earnings and good capital allocation
- Treat steady value creation in equity as supportive

## No-Buy Signals
- Do not buy simply because book value looks large
- Avoid businesses that consume equity without creating strong earnings
- Do not confuse accounting net worth with moat quality

## AI Rules
- Use shareholders' equity as an input to ROE, not as a standalone moat measure
- IF equity grows but returns stay weak THEN lower conviction
- Combine book-value analysis with profitability and leverage checks
